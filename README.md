# UAS - Website Pendaftaran  
**Nama:** Syahid Firmansyah  

## Deskripsi Proyek
Proyek ini merupakan website sederhana berbasis **HTML dan CSS** yang dibuat untuk memenuhi **Ujian Akhir Semester (UAS)**.  
Website ini menampilkan halaman utama dan halaman pendaftaran dengan tampilan sederhana dan mudah digunakan.

## Struktur Folder


## Penjelasan File
- **index.html**  
  Halaman utama website.

- **registration.html**  
  Halaman pendaftaran pengguna (form input data).

- **style.css**  
  File CSS untuk mengatur tampilan dan layout website.

- **assets/images/**  
  Folder yang berisi gambar-gambar pendukung website.

## Teknologi yang Digunakan
- HTML5  
- CSS3  

## Cara Menjalankan Website
1. Download atau clone folder proyek ini.
2. Buka file `index.html` menggunakan browser (Google Chrome, Edge, Firefox, dll).
3. Website siap digunakan.

## Tujuan Pembuatan
Website ini dibuat sebagai:
- Tugas **Ujian Akhir Semester (UAS)**
- Media pembelajaran dasar HTML dan CSS
- Contoh penerapan struktur website sederhana

## Catatan
Proyek ini masih bersifat statis dan belum menggunakan database maupun backend.

## Lisensi
Proyek ini dibuat untuk keperluan akademik dan pembelajaran.
